Project Overview

    Advanced SQL analytics project analyzing YouTube content performance using shared datasets, focusing on engagement quality, creator consistency, and distribution-aware metrics.

Business Questions

    What defines high-quality engagement?    
    Which creators are reliable vs volatile?    
    How does engagement scale with reach?

Datasets

    Video-level metadata and performance    
    Comment-level interaction data

Metrics

    Explicitly defined engagement, intensity, volatility, and percentile-based benchmarks.

SQL Techniques

    CTEs, window functions, conditional aggregation, percentiles, time-series analysis, data validation.

Insights

    Actionable, non-vanity metrics supporting creator and content strategy.

Limitations

    No user-level data    
    No watch-time metrics    
    Comments lack sentiment scoring

Next Steps

    Sentiment analysis
    User cohorting
    Algorithmic ranking simulation
